# Project_IT3180
## PHẦN MỀM QUẢN LÝ THU PHÍ CHUNG CƯ BLUEMOON

### 1. Giới thiệu dự án
Dự án xây dựng phần mềm hỗ trợ Ban quản trị chung cư BlueMoon quản lý thông tin nhân khẩu, hộ khẩu và các khoản thu phí (bắt buộc & tự nguyện) của các hộ gia đình. Thay thế cho việc quản lý thủ công bằng giấy tờ và Excel.

### 2. Công nghệ sử dụng (Tech Stack)
*   **Ngôn ngữ lập trình:** Java (JDK 11 hoặc mới hơn)
*   **Giao diện người dùng (UI):** JavaFX (thiết kế bằng Scene Builder, định dạng `.fxml`)
*   **Cơ sở dữ liệu:** MySQL
*   **Kiến trúc phần mềm:** MVC (Model - View - Controller)
*   **Quản lý phiên bản:** Git & GitHub

### 3. Các chức năng chính (Modules)
1. **Quản lý hộ khẩu/nhân khẩu:** Thêm, sửa, xóa, tách hộ, chuyển khẩu, đăng ký tạm trú/tạm vắng.
2. **Quản lý khoản thu:** Tạo khoản thu mới (Phí vệ sinh, quỹ người nghèo...).
3. **Quản lý nộp tiền:** Ghi nhận số tiền các hộ dân nộp cho từng khoản thu.
4. **Thống kê báo cáo:** Thống kê số tiền đã thu, hộ nào chưa đóng, in danh sách.
